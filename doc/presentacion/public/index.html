Se permite la utilizacion del protocolo HTTPS y la utilización de certificados auto firmados

Notificaciones PUSH

Son mensajes que se envían de forma directa a dispositvios móviles, sirven para notificar de cambios al cliente de una manera rápida y eficiente.

Se utilizó el protocolo SSE (Server sent events) de HTML5.

Más alla de las funcionalidades, el cliente espera calidad y estabilidad

Como garantizarlo?

Proceso de desarrollo

Se deben utilizar herramientas que sean accesibles en cualquier entorno de trabajo.

  1. Elección de una tarea a realizar del sistema de tareas
  2. Resolución de la tarea
  3. Desarrollo de tests
  4. Actualizacion de la documentación
  5. Generación de nuevos tickets
