Descripción detallada

Clase comparadora que implementa un ordenamiento separando el texto por '/' y comparando byte a byte. Es decir, para una clave 't1/t2/.../tn' comparada con 'j1/j2/.../jm', se comparan t1 con j1, t2 con j2, y asi sucesivamente hasta que uno de los dos sea mayor/menor que otro. El orden es inverso que DbComparator, salvo que sigue tomando menor cantidad de miembros como menor clave
